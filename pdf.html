📂 File Tools

Collect PDF & Word files into a 'Combined' folder

Select a main folder – all PDF, DOC, DOCX files from its subfolders will be copied into a new Combined folder inside it.

⚠️ The browser will ask permission to write files inside the folder you choose.

Merge multiple PDFs into one

You can select PDFs from the Combined folder created by the collection tool.

Edit a PDF – rotate, delete, reorder, add text, signature, redact
